## 实施计划

### Phase 1: 地址系统重构 (高优先)
1. **中国省市区完整数据** - 新建 `data/china_regions.dart`，包含全国34省/300+市/2800+区完整三级数据
2. **三级联动选择器** - CupertinoPicker 风格，省→市→区级联
3. **地址编辑页 UI 重构** - 修复对比度/反光问题，使用主题系统颜色

### Phase 2: 订单&发货功能闭环 (高优先)
1. **发货弹窗** - 输入快递公司 + 物流单号，或选择"面对面交付"
2. **物流追踪页** - 展示 LogisticsEntry 时间线
3. **确认收货** - 已在代码中实现（状态变更），加确认弹窗
4. **支付模拟** - 展示支付方式选择弹窗，模拟支付流程
5. **退货申请** - 填写退货原因 → 状态变为 refunding

### Phase 3: 角色权限梳理
1. **admin** = 商家/超级管理员 → 管理仪表盘 + 订单管理 + 商品管理 + 人员管理
2. **operator** = 操作员/店员 → 操作台 + 订单处理
3. **customer** (暂不实现单独登录，但预留)
4. 去除 `admin_dashboard.dart` 中硬编码手机号/用户名，从 authProvider 获取

### Phase 4: 空按钮/空回调全部接通
- order_list: 查看物流、发起退货、删除订单
- order_detail: 查看物流、发起退货
- shop_radar: _viewShopDetail
- admin_dashboard: 仪表盘数据从 orderStatsProvider 获取

### Phase 5: 主题对比度修复
- warning 色调整为 #F59E0B (更深的琥珀色)
- textHint 调整为 #8B95A1 (更深)
- address_edit 页面使用 Theme 系统而非硬编码白底
