## 二、设计方向

### 2.1 复合设计体系: **Aurora Jade（极光玉石）**

融合以下设计风格，打造珠宝行业独特的视觉语言：

| 设计风格 | 应用场景 | 说明 |
|----------|---------|------|
| **Aurora UI 极光流动** | 仪表盘/个人中心背景 | 翡翠绿→深金→暗空蓝渐变动画循环 |
| **Liquid Glass 2.0** | 卡片/面板/浮动栏 | 升级版毛玻璃 + 折射光斑 + 边缘色散 |
| **极简主义** | 商品展示/列表 | 让产品成为焦点，大留白，克制装饰 |
| **新拟物化 (Neumorphism)** | 交互控件 | 按钮/开关/数量选择器，内凹外凸双态 |
| **Bento Grid 便当盒** | 仪表盘/个人中心 | 不规则卡片布局（1×1, 2×1, 2×2） |
| **Motion-driven 动态驱动** | 全局过渡 | 交错进场/数字跳动/脉冲闪烁/液态进度 |
| **深色模式优先** | 全局 | 珠宝奢侈感，OLED 省电 |

### 2.2 色彩升级

```
Aurora 极光色板:
  极光绿:  #2E8B57 → #1A6B3F → #0D4B2A (翡翠渐变)
  极光金:  #D4AF37 → #B8941F → #8B6914 (鎏金渐变)  
  极光蓝:  #0D1B2A → #1B2838 → #2A3A4A (暗空渐变)
  极光紫:  #4A1A6B → #6B2E9B → #8B47CB (紫晶渐变)

折射光效:
  光斑色:  rgba(255, 255, 255, 0.08) → rgba(212, 175, 55, 0.12)
  色散:    彩虹七色 0.03 opacity 边缘渐变
```
