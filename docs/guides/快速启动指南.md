## ❓ 常见问题解决

### Q1: 提示 "No devices found"

**解决方案**:
```bash
# 检查可用设备
flutter devices

# 如果没有 Windows，可能需要启用
flutter config --enable-windows-desktop
```

### Q2: 编译失败，提示 "CMake not found"

**解决方案**:
1. 确认已安装 Visual Studio 2022
2. 确认已安装 "C++ 桌面开发" 工作负载
3. 重启电脑后重试

### Q3: Chrome 版本启动慢

**解决方案**:
```bash
# 使用 release 模式
flutter run -d chrome --release
```

### Q4: APK 安装失败

**解决方案**:
1. 确认手机存储空间充足（需要至少 300MB）
2. 确认已允许安装未知来源应用
3. 卸载旧版本后重新安装

### Q5: 后端启动失败

**解决方案**:
```bash
# 检查 Python 版本
python --version  # 需要 3.8+

# 安装依赖
cd backend
pip install -r requirements.txt

# 重新启动
python main.py
```
