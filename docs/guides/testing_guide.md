## 🔐 本轮高风险回归点（2026-04-08）

- 管理员登录必须填写验证码 `8888`；留空不应放行。
- `logout`、`logout-others`、`reset-password`、`change-password` 后，旧 access / refresh token 必须失效。
- `refresh` 成功后，旧 access token 与旧 refresh token 都不可重放。
- 下单数量必须 `>= 1`；负数和 `0` 应被后端直接拒绝。
- 已取消、超时或争议中的支付单，不可再被后台误确认到账。

## 🔧 常见问题排查

### 登录失败
- 检查输入内容是否正确
- 确认手机号格式 `18925816362`
- 管理员密码 `admin123`，验证码 `8888`
- 操作员密码 `op123456`

### 购物车异常
- 检查本地存储权限
- 尝试清除应用数据重试
- 查看控制台日志

### AI 无响应
- 检查网络连接
- 确认 `DASHSCOPE_API_KEY` 是否有效（见 `lib/config/app_config.dart` `dashScopeApiKeyIssue`）
- 确认设备能访问 `https://dashscope.aliyuncs.com`
- 离线模式应有本地回复

### 页面加载慢
- 检查网络速度
- 图片资源较大时可能需要等待
- 首次加载后应有缓存
